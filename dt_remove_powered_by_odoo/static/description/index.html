Remove “Powered by Odoo”

A lightweight Odoo add-on that removes Odoo branding from login, portal, website footer and POS receipt output while keeping your installation clean and easy to maintain.

Community edition only — no theme overrides required.

What this module removes

  • “Powered by Odoo”, “Manage Databases” and database selector links from login and password reset pages.
  • “Powered by Odoo” branding in the customer portal and website footer.
  • “Powered by Odoo” footer text from printed and emailed POS receipts.
  • No changes to theme templates required.

Features

  1. Remove Odoo branding from login, portal, website footer and POS receipts.
  2. Works without editing your theme templates.
  3. Simple uninstall behaviour with no residual assets.

Installation notes

Install like any standard Odoo module. After installation, clear browser cache and restart the Odoo server so the updated templates take effect.
